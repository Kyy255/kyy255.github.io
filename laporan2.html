KONFIGURASI IP ADDRESS

IP Address

IP Address (Internet Protocol Address) adalah sebuah alamat unik yang digunakan untuk mengidentifikasi perangkat dalam jaringan komputer atau internet. Setiap perangkat yang terhubung ke internet memiliki IP Address agar dapat saling berkomunikasi dan bertukar data.

Jenis-Jenis IP Address

  • IPv4: Menggunakan format angka desimal yang dipisahkan oleh titik, misalnya 192.168.0.1, dengan total 32 bit, menghasilkan sekitar 4,3 miliar alamat unik.
  • IPv6: Menggunakan 128 bit, dengan format angka dan huruf heksadesimal, seperti 2001:0db8:85a3:0000:0000:8a2e:0370:7334, mampu menyediakan jumlah alamat yang jauh lebih banyak dibandingkan IPv4.

Kategori IP Address

  • Public IP Address: Alamat yang digunakan untuk mengidentifikasi perangkat yang dapat diakses melalui internet.
  • Private IP Address: Alamat yang digunakan untuk perangkat di dalam jaringan lokal (rumah/kantor).

Class IP Address

Class Rentang Subnet Mask Penggunaan
Class A 1.0.0.0 - 126.0.0.0 255.0.0.0 Jaringan besar
Class B 128.0.0.0 - 191.255.0.0 255.255.0.0 Jaringan menengah hingga besar
Class C 192.0.0.0 - 223.255.255.0 255.255.255.0 Jaringan kecil

Konfigurasi IP Address Melalui Winbox

Berikut adalah langkah-langkah untuk mengonfigurasi IP Address melalui Winbox:

  • Nyalakan MikroTik dengan menancapkan kabel power ke sumber listrik.
  • Sambungkan kabel LAN (Rj45) ke Ether1 (untuk internet dari ISP) dan Ether2 (ke PC/Laptop).
  • Buka aplikasi Winbox dan akses menu IP > Address.
  • Masukkan IP Address sesuai kebutuhan dan arahkan pada interface yang ingin dikonfigurasi.
  • Klik Apply > OK untuk menyimpan konfigurasi.
