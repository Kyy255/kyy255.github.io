Langkah-langkah Upload Website ke GitHub Pages

Buat akun GitHub

Jika Anda belum memiliki akun GitHub, buat akun di GitHub.

Siapkan repository di GitHub

  • Masuk ke akun GitHub Anda.
  • Klik tombol New Repository di halaman dashboard.
  • Isi nama repository, deskripsi (opsional), dan pastikan memilih opsi "Public".
  • Klik tombol Create Repository.

Persiapkan file website Anda

Pastikan Anda memiliki file HTML, CSS, JavaScript, dan file pendukung lainnya yang ingin Anda upload ke GitHub Pages.

Upload file website ke repository GitHub

  • Buka repository yang telah Anda buat.
  • Klik tombol Add File dan pilih Upload Files.
  • Upload semua file website Anda, termasuk file index.html.
  • Setelah selesai, klik tombol Commit Changes.

Aktifkan GitHub Pages

  • Buka repository Anda.
  • Klik tab Settings.
  • Scroll ke bawah hingga menemukan bagian GitHub Pages.
  • Pada bagian Source, pilih main branch dan folder root.
  • Klik Save dan tunggu beberapa saat hingga website Anda aktif.

Akses website Anda

Setelah GitHub Pages diaktifkan, Anda dapat mengakses website Anda melalui URL berikut:

https://kyy255.github.io/nama-repository/
